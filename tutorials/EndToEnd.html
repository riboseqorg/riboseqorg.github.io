End-to-end Workflow

Brief introduction to the tutorial and its objectives.

This tutorial will guide you through the process of analyzing Ribo-Seq data from start to finish. We will start with the raw data and perform quality control, read alignment, and downstream analysis. By the end of this tutorial, you will have a better understanding of how to analyze Ribo-Seq data and generate meaningful results using RiboSeq.Org.

The tutorial is divided into the following sections, available as separate PDFs:

1. Preprocessing PDF 2. Mapping To Transcriptome PDF 3. Mapping To Genome PDF 4. TCP-Seq and TI-Seq PDF
